## Задание 1

Есть скрипт:

```bash
a=1
b=2
c=a+b
d=$a+$b
e=$(($a+$b))
```

Какие значения переменным c, d, e будут присвоены? Почему?

| Переменная  | Значение | Обоснование |
| ------------- | ------------- | ------------- |
| `c`  | a+b  | Так как a и b без символа $, обращения к переменным нет   |
| `d`  | 1+2  | Проивзводится обращение к переменным, конкатенация их строковых значений |
| `e`  | 3  | Используются скобки значит арифметические операции |

## Задание 3

Необходимо написать скрипт, который проверяет доступность трёх IP: `192.168.0.1`, `173.194.222.113`, `87.250.250.242` по `80` порту и записывает результат в файл `log`. Проверять доступность необходимо пять раз для каждого узла.

### Ваш скрипт:

```bash
declare -i i=1
declare -i count=5
while (($i<=count))
do
  for host in 192.168.0.1 173.194.222.113 87.250.250.242; do
    nc -w1 -zv $host 80
    echo host $host status $? time: `date` >> status.log
  done
  i+=1
done
```

### Ваш скрипт:

```bash
# проверял на локальном nginx (так как есть возможность включать и выключить сервис) поэтому здесь используется 127.0.0.1, для себя оставил так, если принципиально то поставлю хосты как в задании
while true

  for host in 173.194.222.113 87.250.250.242 127.0.0.1; do
    nc -w1 -zv $host 80
    if (($?!=0))
    then

    else
      echo host $host status $? time: `date` >> status.log
```
